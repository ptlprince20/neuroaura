# Neuro-aura: Total Mastery Learning OS 🧠✨

**Neuro-aura** is a high-fidelity, professional-grade Learning Operating System (LOS) designed specifically for high-achieving Computer Science students. It bridges the gap between raw data and cognitive insight, providing a "Mastery-as-a-Service" environment.

---

## 🚀 The Vision
In an era of information overload, Neuro-aura prioritizes **Cognitive Efficiency**. By tracking your neural load, processing speed, and memory retention, the OS crafts a tailored Roadmap that evolves with your brain's performance.

## 💎 Elite Features

### 1. **Neural Mastery Path (The Roadmap)**
A multi-layered interactive graph covering the four pillars of modern CSE: **Operating Systems**, **Database Management**, **DSA**, and **Python Programming**.
- **Real-time Metrics**: View mastery percentage and completed modules directly on the node.
- **Video Session Integration**: Trigger high-fidelity video tutorials directly inside the OS with topic-specific focus.

### 2. **Scientific OS Analytics (Cognitive DNA)**
The dashboard monitors your **Cognitive DNA** in real-time:
- **Style**: Visual-Spatial, Auditory, or Kinesthetic modeling.
- **Speed**: Calculation of your processing capacity in Ops/m.
- **Load**: A dynamic 1-10 scale measuring your current mental fatigue.
- **Streak Tracker**: A 7-day milestone system with glowing visual indicators of your consistency.

### 3. **The Action Hub (The Command Center)**
A dedicated quadrant for rapid skill verification:
- **🚀 Challenge Mode**: Intense randomized testing with selectable difficulty (Easy, Intermediate, Hard).
- **📔 Quiz Review**: Direct analysis of past errors with AI-driven suggestions for improvement.
- **📼 Video Review**: Comprehensive watch-history log to keep your visual learning on track.
- **🔍 Topic Quiz**: Pinpoint testing where you define the exact subject depth.
- **📤 Export Report**: Instantly download a professional PDF mastery report for offline review.

### 4. **Aura AI Assistant 🤖**
Your personal neural guide. Accessible via a floating interface, Aura provides contextual help, OS navigation tips, and motivational cognitive insights.

### 5. **Premium UI/UX (Aura-X Elite)**
- **Dual-Mode Theming**: Seamlessly switch between **Midnight Obsidian (Dark)** and **Crisp Slate (Light)** themes.
- **Glassmorphism Design**: High-end frosted glass aesthetics with hairline borders.
- **Cinematic Gateway**: A professional preloader sequence upon login for a premium application feel.

---

## 🛠️ Technical Architecture
- **Framework**: Python / Flask
- **Backend**: SQLAlchemy & SQLite (Autonomous database seeding)
- **Frontend**: ES6 Modern JavaScript, Vanilla CSS3 (Custom Design System)
- **Automation**: Integrated PDF rendering via `html2pdf.js`

## ⚙️ Quick Start
1. **Initialize Environment**: `pip install flask flask-sqlalchemy`
2. **Launch OS**: `python app.py`
3. **Link Neural Core**: Visit `http://127.0.0.1:6060`
   - *Use `demo`/`demo123` or **Guest Mode** for instant access.*
